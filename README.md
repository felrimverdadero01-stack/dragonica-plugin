# ドラゴニカ対戦ログ集計ツール

対戦ログページの各キャラクターの行動を自動で集計し、見やすい表にまとめるTampermonkeyスクリプトです。

## 機能

### 集計情報

各キャラクターについて、次の情報を集計します：

#### ターン別情報（表で表示）
- **ターン番号** - 各ターン
- **与えたダメージ** - 対象別のダメージ合計
- **与えたHP回復** - 対象別のHP回復合計
- **与えたSP回復** - 対象別のSP回復合計
- **ステータス変化** - ステータスの上昇/低下値
- **最大HP増加** - 最大HPの増加値

#### 合算値（サマリー）
- **総与ダメージ** - 全ターンの与ダメージ合計
- **総HP回復** - 全ターンのHP回復合計
- **総SP回復** - 全ターンのSP回復合計
- **総最大HP増加** - 全ターンの最大HP増加合計

## インストール方法

### 1. Tampermonkeyのインストール
お手持ちのブラウザに応じて、以下からTampermonkeyをインストールしてください：

- Chrome: https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobela
- Firefox: https://addons.mozilla.org/firefox/addon/tampermonkey/
- Safari: https://www.tampermonkey.net/
- Edge: https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmleppure

### 2. スクリプトのインストール
1. 本フォルダの `battlelog_analyzer.user.js` ファイルの内容をコピーします
2. Tampermonkeyの「+ユーザースクリプト」（または新規スクリプト作成）を選択します
3. コピーした内容を貼り付け、保存（Ctrl+S）します

### 3. 使用
対戦ログページ（例：`https://metropolis-c.sakura.ne.jp/teikigame/dragon1st/battlelogs/....html`）にアクセスすると、自動的に集計結果がページ最下段に表示されます。

## ページ構造

### 対応ページのURL形式
```
https://metropolis-c.sakura.ne.jp/teikigame/dragon1st/battlelogs/*.html
```

### 集計対象
- ✅ 味方チーム全キャラクター
- ✅ 敵チーム全キャラクター
- ✅ すべてのターン

### 同名キャラクターの処理
味方チームと敵チームに同名のキャラクターがいる場合、自動で区別して集計します。

## 集計対象となるアクション

### ダメージ
```
〇〇 の「××」が △△ に ★★ ダメージを与えた
```

### HP回復
```
〇〇 の「××」が △△ を ★★ 回復した
```

### SP回復
```
〇〇 の マナ！ 〇〇 の SPが ★★ 回復した
```

### ステータス変化
```
〇〇 の ステータスが上昇！
ATK +5 ...

〇〇 の ステータスが低下！
ATK -5 ...
```

### 最大HP増加
```
〇〇 の最大HPが +★★ 増加した
```

## 表示形式

### 名前別セクション
各キャラクター毎に以下が表示されます：

1. **キャラクター名**（セクションタイトル）
2. **ターン別集計表**
   - ターン | ダメージ | HP回復 | SP回復 | ステータス | 最大HP増加
3. **合算値サマリー**
   - 総与ダメージ
   - 総HP回復
   - 総SP回復
   - 総最大HP増加

### カラー分け
- **味方チーム**: 青系（#0066cc）
- **敵チーム**: 赤系（#cc0000）

## トラブルシューティング

### 集計が表示されない場合
1. ブラウザのコンソール（F12 → Console）でエラーを確認
2. ページが完全に読み込まれているか確認
3. URLが対応ページであるか確認

### 集計結果が不完全な場合
- ログのテキスト形式が想定と異なる可能性があります
- ブラウザ開発者ツール（F12）でページのHTMLを確認してください
