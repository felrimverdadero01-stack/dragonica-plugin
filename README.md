# ドラゴニカ対戦ログ集計ツール

対戦ログページの各キャラクターの行動を自動で集計し、見やすい表にまとめるTampermonkeyスクリプトです。

## 機能

### 集計情報

各キャラクターについて、次の情報を集計します：

#### ターン別情報（表で表示）
- **ターン番号** - 各ターン
- **与えたダメージ** - 対象別のダメージ合計
- **与えたHP回復** - 対象別のHP回復合計
- **与えたSP回復** - 対象別のSP回復合計
- **ステータス変化** - ステータスの上昇/低下値
- **最大HP増加** - 最大HPの増加値

#### 合算値（サマリー）
- **総与ダメージ** - 全ターンの与ダメージ合計
- **総HP回復** - 全ターンのHP回復合計
- **総SP回復** - 全ターンのSP回復合計
- **総最大HP増加** - 全ターンの最大HP増加合計
